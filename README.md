# Student Performance Analysis & Prediction

## Overview
This project analyzes student academic records and predicts student performance based on various features such as study time, parental education, and previous scores.

## Key Tasks
- Data cleaning
- Exploratory data analysis (EDA)
- Feature relationship visualization
- Performance prediction

## Techniques Used
- Pandas & NumPy
- Data Visualization (Matplotlib/Seaborn)
- Machine Learning (Regression/Classification)

## Outcome
The project identifies important factors affecting student performance and predicts future academic results.
